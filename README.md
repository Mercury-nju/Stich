# Stitch / 长图拼接

A simple iOS app to combine multiple photos into a single vertical long image.

一款简单的 iOS 应用，将多张照片合并成一张垂直长图。

## Features / 功能

- Select multiple photos from your library / 从相册选择多张照片
- Drag and drop to reorder / 拖拽排序
- Preview before saving / 保存前预览
- Save to photo library / 保存到相册
- Privacy first - all processing happens locally / 隐私优先 - 所有处理都在本地完成

## Requirements / 要求

- iOS 17.0+
- Xcode 15.0+
